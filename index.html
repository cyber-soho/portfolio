Teaching Philosophy

Centered on the principle of "Learning by Doing."

Project-Based Learning

Courses are structured around real-world projects to build practical problem-solving skills.

Industry Relevance

The curriculum is dynamic, replacing obsolete material with in-demand skills like cloud and automation.

Strong Industry Partnerships

Advocating for partnerships to provide internships, modern lab equipment, and networking opportunities.

Accessible Learning

Emphasis on free and accessible online resources like interactive labs and video tutorials.

Mentorship over Lecturing

Fostering a collaborative environment where students receive individual attention and guidance.

Experience-First Certifications

Guiding students to build a solid practical foundation before pursuing industry certifications.

Proposed Curriculum

A robust foundation in IT with a focus on hands-on labs and modern tech.

Module 1: IT Hardware Fundamentals

This module focuses on understanding the core components of modern computing systems, from desktops to servers.

  • Courseware (Paid): CompTIA TestOut + LabSim
  • Courseware (Free): LinkedIn Learning, O'Reilly/Safari (via library subscription)

Module 2: Operating Systems

Client OS
  • Microsoft Windows 11
  • Linux (Ubuntu, Mint, Kali, Parrot)
  • Unix (macOS) & Mobile (iOS/Android)
Server / Network OS
  • Windows Server 2019
  • Red Hat Enterprise Linux (via CentOS/Alma/Rocky)
  • Debian
Courseware
  • Paid: CompTIA, Red Hat Academy
  • Free: Microsoft Learn, LinkedIn Learning

Module 3: Networking Fundamentals

Building a strong, vendor-neutral understanding of networking principles is critical before tackling specialized certifications. This module covers the essentials.

  • Topics: OSI Model, TCP/IP, Subnetting, Routing, Switching, Wireless Networking.
  • Courseware (Free): Cisco NetAcad - IT Essentials
  • Courseware (Paid): CompTIA Network+

Module 4: Scripting & Automation Foundations

Languages & Formats
  • PowerShell & Bash
  • Python (with C fundamentals)
  • XML, YAML, JSON & Markdown
Courseware
  • A wide variety of free and paid online platforms like Codecademy, Udemy, and library resources.

Module 5: Modern DevOps & Cloud Infrastructure

  • Virtualization: Hosted (VirtualBox) & Native (KVM, ESXi)
  • Containers & Orchestration: Docker/Podman, Kubernetes/OpenShift
  • Cloud Platforms: AWS, Azure, GCP
  • Automation: Ansible (Config Mgmt), Terraform (Infra as Code)
  • Courseware: Platform-specific free tiers (AWS Educate) and Red Hat Academy.

Module 6: Cybersecurity Principles

This module introduces the foundational domains of cybersecurity, preparing students for specialized roles.

  • Domains: Offensive Security (Ethical Hacking), Defensive Security (Blue Teaming), IT Governance & Legislation
  • Courseware (Paid): EC-Council, Hack The Box, TryHackMe, CompTIA Security+
  • Courseware (Free): Online library resources and platform-specific training.
